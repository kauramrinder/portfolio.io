My Projects
Face Recognition System
Description

Face Recognition Sysytem is one of the most biometric information processes, its applicability is easier and working range is larger than others such as fingerprint, iris scanning,signature,etc. The system uses a combination of techniques in two topics, face detection and recognition. The face detection is performed on live acquired images. Processes utilized in the system are cropping and resizing of the acquired image, detecting the face, applying filter upon the detected part, facial feature extraction on a face candidate. then a face classification method that uses K-Nearest Neighbor is integrated in the system. The system is then trained and then tested with a database generated with multiple images of different persons. The tested system has accpetable performance to recognize faces within intended limits.

System Software

MATLAB R2012a

Database used to check accuracy rate

Japanese Female Facial Expression (JAFFE) Database

Future Scope

Face recognition remains as an unsolved problem and a demanded technology. Therefore there are many such fields in this project where we can work further to make our system more reliable and much more compact.

Firstly, we can make our system further accurate using a strong feature extraction algorithm. There are several algorithms available but we have to choose that one which implements the sysytem with better results increasing the level of accuracy. We can also use combination of algorithms to make our approach of recognition robust enough.

Secondly, we can work further in the classification area using better classifiers to reach the actual goal of our system that is it will show desired and accurate recognition results invariant of environment and angle of pose to boost up the performance of our system.
