Leave Application

Github Link

Description

Leave Application is a basic Desktop Application designed with Java using Eclipse. This basically is designed to be used by the employees of a particular company, helping the company to track the leaves taken by its employees. Company can monitor leaves in two ways: (1) How many employees have taken leave for a particular day/week, (2) How many callouts a particular employee has taken. This application also helps managers to accept or decline the leaves taken by other employees and update the database accordingly.

Front-end Technology

Java using Eclipse

Backend Technology

Oracle Database

Future Scope

This desktop application can be improved to calculate the total number of hours worked per week and therefore the weekly pay can be calculated for a particular employee.Also, the use of this application be braodened by the increasing number of comapanies using it.
